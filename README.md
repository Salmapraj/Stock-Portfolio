# Stock Portfolio App

A stock portfolio management application built with React and TypeScript.  
Users can add, edit, delete and filter stocks in their portfolio.

## Features

- Add stock
- Edit stock
- Delete stock
- Search stocks
- Filter by price
- Local storage persistence

## Data Source

- Stock data is stored in static JSON files
- No backend or API is used
- Data is stored in browser localStorage

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- HighCharts
- Shadcn UI
- Vite

## Installation

1. Clone the repository

```bash
git clone https://github.com/Salmapraj/Stock-Portfolio.git
```

2.  Navigate to project folder
- cd stock-portfolio

3. Install dependencies
- npm install

4. Run development server
- npm run dev

5. Open in browser
- http://localhost:5173
